WebAR Demo

With modern smartphones it is now possible to deliver performant AR experiences from the browser using web technologies.

If you combine this with QR codes you can deliver cross-browser AR experiences with very low user friction.

Example use cases:

  • A customer walks into a museum, besides each artifact is a QR code, scanning the QR code brings up a 3D animation in AR of the artifact.
  • A shopper sees a mannequin with a cool jacket, she scans a QR code which launches an AR experience of the jacket but with the ability to select other colors. Maybe the AR experience helps set the mode my showing snow falling on the jacket.

In each example above the user never has to install an app and never has to type in a URL. On iOS and Android a QR scanner is now built in to the stock camera app.

Retail Demo

Step 1

Set up a mannequin or clothes rack with sample jacket. Print out this QR code and place it next to the jacket.

Step 2
